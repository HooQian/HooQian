# HooQian

## 由来

&emsp;&emsp;先来看一些简单的单词：Look、Book、Good……

&emsp;&emsp;找一找共同点？

&emsp;&emsp;Yes，英文中“oo”的发音和拼音中的“u”几乎一样。

&emsp;&emsp;So，Hoo ≈ Hu

## 意义

&emsp;&emsp;这可以用来代替同音的姓氏，如胡、虎、呼、忽、狐、扈、户、鄠……

&emsp;&emsp;如果很幸运，你名叫前、钱、千、牵、浅、签、欠、铅、嵌、钎、迁、钳、乾、谴、谦、潜、歉、纤、扦、遣、黔、堑、仟……
